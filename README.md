# Fundamentos de Java para Principiantes 🚀

Curso introductorio de programación en **Java** dirigido a estudiantes de
secundaria (12 a 17 años) **sin experiencia previa**. Todo el curso se sigue
desde el navegador: **no necesitas instalar nada**.

Este repositorio contiene el **código fuente** de las prácticas y el
**plan de curso** completo.

## ¿Qué necesito para empezar?

Solo un **navegador web** (Chrome, Firefox, Safari o Edge) y conexión a
internet. Las prácticas se resuelven en editores de código en línea y gratuitos:

- [Replit](https://replit.com)
- [JDoodle](https://www.jdoodle.com/online-java-compiler/)
- [OnlineGDB](https://www.onlinegdb.com/online_java_compiler)

No hace falta crear una cuenta para ejecutar el código.

## ¿Cómo uso este repositorio?

1. Mira el video de la lección en YouTube (enlaces en el plan de curso).
2. Abre el editor en línea de la práctica (enlace en cada lección).
3. Abre el archivo de la carpeta [`starter-files/`](starter-files) que
   corresponde a la lección, copia su contenido al editor y completa los `TODO`.
4. Si te trabas, compara tu solución con la de
   [`completed-examples/`](completed-examples).

## Contenido de las lecciones

El curso son **5 lecciones** (un video por lección).

| #  | Lección | Temas | Carpeta |
|----|---------|-------|---------|
| 1  | Primeros pasos en Java | Qué es programar y Java, Hola Mundo, variables, tipos y operadores | `lesson01-primeros-pasos` |
| 2  | Interacción y decisiones | Entrada con `Scanner`, `if`/`else`, `switch` | `lesson02-interaccion-decisiones` |
| 3  | Repetir y reutilizar | Bucles `for`/`while`, métodos y parámetros | `lesson03-bucles-metodos` |
| 4  | Programación Orientada a Objetos | Clases, objetos, constructores, encapsulación (getters/setters) | `lesson04-poo-clases` |
| 5  | Proyecto final y buenas prácticas | Arrays de objetos, proyecto "Registro de Estudiantes", errores comunes | `lesson05-proyecto-final` |

## ¿Cómo ejecuto el código en el editor en línea?

1. Entra a cualquiera de los editores recomendados.
2. Pega el contenido de un archivo `Main.java`.
3. Presiona **Run**.
4. Para las lecciones con `Scanner`, escribe los datos en el panel de
   **Input / stdin** del editor.

> 💡 El equipo del curso puede compilar y probar los ejemplos en su computadora
> con los scripts de [`scripts/`](scripts) (requiere tener el JDK instalado).
> Esto es **solo para el equipo**; los estudiantes no lo necesitan.
